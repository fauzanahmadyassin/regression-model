Define the Neural Network model structure

  1. Input layer with 11 neurons
  2. First hidden layer with 12 neurons
  3. Second hidden layer with 8 neurons
  4. Output layer with single output
  5. Activation function used are Rectifier and Linear Activation Function for generates output
  6. Optimizer used is Adamax with 0.001 learning rate
  7. Epoch number is 100
  8. Batch size is 50
In [15]:
from keras import optimizers

adam = optimizers.Adam(lr=0.001)

test_split = 0.2

model = Sequential()
model.add(Dense(12, input_dim=11, kernel_initializer='normal', activation='relu'))
model.add(Dense(8, activation='relu'))
model.add(Dense(1, activation='linear'))

model.compile(loss='mse', optimizer='adam', metrics=['mse','mae'])  # mse - mean squared error, mae - mean absolute error

classifier = model.fit(X_train, y_train, epochs=100, batch_size=50,  verbose=1, validation_split=test_split)
